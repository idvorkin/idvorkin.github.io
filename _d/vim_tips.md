### Code Changes

- <space>rn - Rename
- yaf (yank a function) - Via TreeSitter+TreeSitter Motions
- \xX -> Open Diagnostics - Via Trouble
- <space>ai auo import
- <space>code action
- <space>gd - go to definition - Via LSP
- <space>gt - go to type
- \cc - Comment or uncomment

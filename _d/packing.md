A durable, **trip-agnostic** packing checklist. This is the _source_ — every specific trip gets a trimmed copy: take the relevant sections, prune to the trip, decide the per-trip choices (cameras, magic props, kettlebells). Don't pack from this directly — pack from the trimmed trip file.

It's an **items + decisions** list. The _process_ of taking time off — when to pack, airport timing, "Walmart-it if you forget," the peak-end rule — lives at [/time-off](/time-off).

Many sections split **Road trip** (car space — heavy/bulky OK) vs **Airplane** (carry-on-safe, weight/liquid limits).

## Magic

Decide which props per trip — don't default to all of them.

- [ ] **Decks** — 2 decks per 3 days (scale up for longer)
- [ ] **Mem deck** (Smart Hands / card overlay track) — written/marked copy too
- [ ] **Coins** — pick the set
- [ ] **Rubber bands — two colors** (band magic / crazy man's handcuffs)
- [ ] **Balloons** — pump + stash of 260s, **and pre-make a batch before the trip** so you're not building from scratch on the spot
- [ ] Sponge balls — **default: leave** (listed so it's a conscious skip, not a forget)
- [ ] Reload cards + sleeves, paper clips
- [ ] Close-up case / pad if performing beyond family

## Physical Health

### Sleep

- [ ] Eye mask
- [ ] Earplugs
- [ ] Sleepwear

### Exercise

Minimum viable gear. The McGill Big 3 needs **zero** gear — no excuse on the trip.

**Always (any trip)**

- [ ] Peanut ball + giant lacrosse ball (back maintenance)
- [ ] Resistance bands — one light, one heavy (door-hinge anchor)
- [ ] Running shoes (even if "not going to run" — you will)
- [ ] Sweat-wicking shorts + shirt

**Road trip only (car space)**

- [ ] Kettlebells (keep in the car; don't fly with them), gripper, heavy club, pull-up door bar

**Airplane** — stick to the "Always" carry-on-safe kit.

### Meds

- [ ] **Prescription continuity** — don't run out mid-trip; plan refills ahead and pack doses appropriately
- [ ] Sleep aid — for long flights
- [ ] Decongestant — flight ear pressure
- [ ] First aid: ibuprofen, band-aids, kid staples

### Glasses / eyes

- [ ] Everyday glasses + sunglasses
- [ ] Reading glasses — one pair
- [ ] Distance glasses — only if seeing a show
- [ ] Glasses wipes

### Hygiene

- [ ] Toiletries dupe kit (permanent grab-and-go — the perennial forget)
- [ ] Toothbrush / paste / floss, deodorant, razor

### Food / fuel

**Always**

- [ ] Power bars — 1/day + 2 extras (keep them accessible, not in the trunk)
- [ ] Water bottle
- [ ] Snacks (drive or flight)

**Road trip only**

- [ ] Popcorn popper, instant soup, travel scale, cooler

## Tech

- [ ] **Kindle Scribe** (journaling — default yes)
- [ ] iPad / laptop (as needed)
- [ ] Webcam (for magic)
- [ ] Travel keyboard
- [ ] Watch charger
- [ ] Phone charging cables ×2 (kids steal yours)
- [ ] Power banks — one big (20,000 mAh), one small (10,000 mAh)
- [ ] Three-outlet USB-C multi-plug
- [ ] AirPods ×2
- [ ] **AirTags** on everything (luggage + bags)
- [ ] Hotspot fallback if cabin/venue wifi is unknown (esp. scheduled calls)

**Road trip**

- [ ] **Travel router** — pre-configure to bypass flaky captive portals. (Captive-portal check: `http://captive.apple.com/hotspot-detect.html`)

### Cameras (decide per trip — don't pack all of them)

| If the trip is…                  | Bring                                   |
| -------------------------------- | --------------------------------------- |
| Family / walk-around / easy vlog | **Pocket gimbal cam** (default one-cam) |
| Action / water / POV             | **360 cam** + mounts + extra SD cards   |
| Scenic / landscapes / aerials    | **Drone** (check local rules first)     |

- [ ] Pocket gimbal cam — default if bringing only one
- [ ] 360 cam — pre-mount, charge, horizon-lock before departure
- [ ] Drone — confirm legal at destination + airline battery rules (carry-on only, watch Wh limits)
- [ ] Selfie sticks — a tall one (overhead reach) + a pocket one (invisible-stick shots)
- [ ] Extra SD cards + spare batteries + chargers for whatever's chosen

## Clothing & Luggage

### Always

- [ ] Underwear ×7, socks ×4
- [ ] Shirts ×5
- [ ] Long-sleeve / sweatshirts ×7, fleece ×1
- [ ] Rain jacket ×1, pants ×1
- [ ] Shoes: comfortable walkers + running shoes (waterproof option for wet climates)
- [ ] Swim trunks + goggles
- [ ] Foldable hat, rain hat, warm fleece cap
- [ ] Sunscreen

### Gym clothes

- [ ] Shorts / sweatpants
- [ ] Lock for gym lockers

### Winter / cold trips only

- [ ] Warm layer — cabins/water run colder than forecast
- [ ] Base layers, gloves, beanie, heavier coat, hand warmers

### Luggage

- [ ] Compression straps
- [ ] **Check the extra suitcase pocket** — easy to lose bulky items in there

## Documents & Money

- [ ] Passport + any required immigration/travel documents
- [ ] Trusted-traveler card (Global Entry / Nexus)
- [ ] Wallet: ID, credit cards
- [ ] Trip confirmations + lodging specifics (entry code, address, host) saved **offline**

## Downtime / Comfort

- [ ] Audiobooks (drive days)
- [ ] Reusable bag / daypack

## Trip-type add-ons (pull only the relevant block)

- **Beach / water**: towels, water shoes, dry bag, extra sunscreen.
- **Cold / snow**: see Clothing → Winter.
- **International**: passport + travel docs, adapters, currency, eSIM/roaming, any required visas/vaccination records.
- **Camping / cabin off-grid**: headlamp, power bank, bug spray, layered sleep system, backup food.
- **Conference / work travel**: laptop + dongles, presentable clothes, notebook.

## The "Day 2" forget-traps

The items most likely to bite on day 2 — answer these per trip:

1. **Back maintenance** — peanut ball + lacrosse ball in the bag? Costs nothing.
2. **Magic specifics** — decks / coins / balloons (pre-made?) decided?
3. **Rain layer + waterproof shoes** for wet weather?

- **Travel process & timing** — pack the afternoon before, Walmart-it if you forget, airport-early, the peak-end rule: [/time-off](/time-off).

Keep the list **lean**: when a trip surfaces a new reliable forget-trap, add it so the next trip inherits it; when an item stops being a problem, cut it.
